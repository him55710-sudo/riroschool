# Product Requirements Document (PRD)

## 1. 개요
본 애플리케이션은 한국 교육 멘토 워크플로우를 위한 "포트폴리오/보고서" PDF 생성 웹 서비스입니다. 단일 주제 입력을 기반으로 고품질의 보고서를 자동 생성합니다. 

## 2. 주요 기능
- **단일 주제 입력**: 사용자가 주제를 입력하면 AI가 목차를 구성하고 내용을 생성합니다.
- **페이지 분량 티어**:
  - FREE: 1–10 페이지
  - PAID: 11–20 페이지
  - PAID: 21–30 페이지
- **AI 기반 생성**: 서버 측 Gemini API를 사용하여 출처가 명시된 고품질 레이아웃의 보고서를 생성합니다.
- **제출 보조 기능**: 서드파티 플랫폼 자동 제출(매크로) 기능은 규정/위험 문제로 제외되며, 대신 '클립보드 복사' 및 '선택적 폼 자동완성'과 같은 보조 기능만 제공합니다.

## 3. 로드맵 (3-Step)
- **Step 1**: 무료 1~10페이지 포트폴리오 생성 및 엔드투엔드 PDF 다운로드 구현
- **Step 2**: 페이월(Paywall), 결제 연동, 크레딧 시스템 도입
- **Step 3**: 품질 향상 (인용구, 다이어그램, 레이아웃 개선, 중복 제거, 웹 검색 기능 강화)
